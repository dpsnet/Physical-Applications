# 统一场理论物理应用

本仓库包含统一场理论的物理应用和实验验证，专注于P系列的理论模块。

## 仓库结构

### 物理应用
- **P-1**：测度场、多重扭转与规范理论的统一场理论
- **P-2**：Dirac算子与重整化群
- **P-3**：多重分形与弦理论
- **P-4**：全息对偶与量子化
- **P-5**：暗物质理论
- **P-6**：时空硬化效应理论
- **P-7**：微观时间反转实验与时间单向性
- **P-8**：黑洞理论
- **P-9**：核聚变理论
- **P-10**：圈量子理论与分形克利福德代数兼容统一
- **P-11**：固定拓扑-动态谱维范式下的宇宙起源
- **P-12**：量子力学基本现象的几何解释
- **P-13**：宇宙纤维自旋的多重扭转理论解释
- **P-14**：黑洞遗迹后的宇宙演化与高维宇宙理论
- **P-15**：多元宇宙物理规律与跨宇宙交互理论
- **P-16**：光子频率与粗粒量子化理论
- **P-17**：正交扭转的几何本质
- **P-18**：扭转演化动力学与谱维度流动
- **P-19**：正交扭转理论与拓扑保护机制
- **P-20**：红移的几何统一理论
- **P-21**：强相互作用的麦克斯韦型方程组
- **P-22**：粘性流体流动的多重扭转计算理论

### 实现文件
- **源代码**：物理模拟的Python实现
- **数据**：实验数据与模拟结果
- **图表**：物理现象可视化与理论预测

### 语言版本
本仓库中的所有论文均提供中英文双语版本：
- **中文版**：`_cn.md` 扩展名的文件
- **英文版**：`_EN.md` 或 `_en.md` 扩展名的文件
