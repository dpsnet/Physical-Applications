## 2. 分形晶格上的电子结构

### 2.1 Sierpinski carpet上的能带结构

Sierpinski carpet是具有分形维数 $d_f = \ln 8 / \ln 3 \approx 1.893$ 的经典分形结构。考虑紧束缚模型：

$$H = -t \sum_{\langle i,j \rangle} (c_i^\dagger c_j + h.c.)$$

其中求和仅针对Sierpinski carpet上的最近邻格点。

**定理2.1.1（分形能带定理）** Sierpinski carpet上的电子能带表现出分形能谱，能级分布满足幂律关系：

$$N(E) \sim E^{d_f/2}$$

其中 $N(E)$ 为态密度，$d_f$ 为分形维数。

**证明**：

1. 由分形空间中的薛定谔方程：
   $$-\frac{\hbar^2}{2m} \nabla^2 \psi(x) + V(x)\psi(x) = E\psi(x)$$
   其中 $V(x)$ 为分形晶格势。

2. 利用分形测度的尺度变换性质，本征值满足：
   $$E_n \sim n^{2/d_f}$$

3. 态密度为：
   $$N(E) = \sum_n \delta(E - E_n) \sim E^{d_f/2 - 1}$$

4. 因此累积态密度：
   $$N(E) = \int_0^E N(E') dE' \sim E^{d_f/2}$$ ∎

**推论2.1.1（分形van Hove奇点）** 分形晶格的态密度在带边表现出幂律奇异性：

$$N(E) \sim |E - E_c|^{d_f/2 - 1}$$

而非整数维数的van Hove奇点。

### 2.2 分形维数与电导率的关系

**定理2.2.1（分形电导标度律）** 在分形晶格上，直流电导率随系统尺度 $L$ 的标度行为为：

$$\sigma(L) \sim L^{2 - d_s}$$

其中 $d_s$ 为谱维数，与分形维数 $d_f$ 的关系为 $d_s = 2d_f/(2+\theta)$，$\theta$ 为谱维指数。

**证明**：

1. 由爱因斯坦关系：
   $$\sigma = e^2 N(E_F) D$$
   其中 $D$ 为扩散系数。

2. 在分形结构上，扩散是反常的：
   $$\langle r^2(t) \rangle \sim t^{2/d_w}$$
   其中 $d_w$ 为随机游走维数。

3. 扩散系数与尺度相关：
   $$D(L) \sim L^{2-d_w}$$

4. 谱维数定义为：
   $$d_s = 2d_f/d_w$$

5. 因此：
   $$\sigma(L) \sim L^{d_f} \cdot L^{2-d_w} = L^{2-d_s}$$ ∎

**数值结果**：
- Sierpinski gasket: $d_f = \ln 3/\ln 2 \approx 1.585$, $d_s \approx 1.365$
- Sierpinski carpet: $d_f \approx 1.893$, $d_s \approx 1.80$

对于Sierpinski carpet，理论预言：

$$\sigma(L) \sim L^{0.20}$$

表明电导率随系统尺度缓慢增长，这是分形几何的特有效应。

### 2.3 分形色散关系与反常输运

**定理2.3.1（分形色散关系）** 分形晶格上的低能有效色散关系为：

$$E(k) \sim |k|^{d_w/2} = |k|^{d_f/d_s}$$

而非通常的 $E \sim k^2$。

**证明**：

由量纲分析，在分形结构中：
- 能量量纲: $[E] = L^{-d_w/2}$
- 动量量纲: $[k] = L^{-1}$

因此：
$$E \sim k^{d_w/2} = k^{d_f/d_s}$$ ∎

**物理意义**：
- 对于Sierpinski carpet: $E \sim k^{1.05}$，接近线性色散
- 这种准线性色散导致类似狄拉克费米子的输运性质
- 电子有效质量在分形结构上变为尺度依赖

## 3. 拓扑绝缘体的扭转解释

### 3.1 表面态的几何起源

拓扑绝缘体的关键特征是体带隙中存在的受拓扑保护的表面态。从分形-扭转理论的角度，这可以重新解释为：

**定理3.1.1（表面态的几何起源定理）** 拓扑绝缘体的表面态起源于分形Clifford代数中的扭转结构，表面态的稳定性由扭转拓扑不变量保证。

**证明**：

1. 考虑三维拓扑绝缘体的有效哈密顿量：
   $$H = \hbar v_F \vec{k} \cdot \vec{\sigma} + m(k)\sigma_z$$

2. 引入分形扭转后，Dirac算子修正为：
   $$D_{eff} = \gamma^\mu (\partial_\mu + \tau \partial_\mu \ln \mu(x))$$

3. 在表面（$z=0$），边界条件由扭转决定：
   $$(i\gamma^\mu n_\mu + \tau \ln \mu) \psi|_{surf} = 0$$

4. 这保证了表面态的存在，其波函数为：
   $$\psi_{surf}(k_x, k_y, z) = \psi_0 e^{-\lambda z} e^{i(k_x x + k_y y)}$$
   其中衰减长度 $\lambda^{-1} \sim \tau^{-1}$ ∎

### 3.2 陈数的扭转修正

**定理3.2.1（扭转陈数修正）** 在分形-扭转理论中，陈数获得如下修正：

$$\tilde{C}_1 = C_1 + \frac{\tau^2}{4\pi^2} \int_{BZ} d^2k \, \text{Tr}[F_{xy}] \cdot \mathcal{R}(k)$$

其中 $\mathcal{R}(k)$ 为分形曲率，$\tau$ 为扭转强度。

**推导**：

1. 标准陈数定义为：
   $$C_1 = \frac{1}{2\pi} \int_{BZ} \mathcal{F}$$
   其中 $\mathcal{F} = d\mathcal{A} + \mathcal{A} \wedge \mathcal{A}$ 为Berry曲率。

2. 在分形Clifford代数中，联络获得扭转贡献：
   $$\mathcal{A}_{\mu} \to \mathcal{A}_{\mu} + \tau \mathcal{T}_{\mu}$$

3. Berry曲率修正为：
   $$\tilde{\mathcal{F}} = d(\mathcal{A} + \tau \mathcal{T}) + (\mathcal{A} + \tau \mathcal{T}) \wedge (\mathcal{A} + \tau \mathcal{T})$$
   $$= \mathcal{F} + \tau(d\mathcal{T} + \mathcal{A} \wedge \mathcal{T} + \mathcal{T} \wedge \mathcal{A}) + \tau^2 \mathcal{T} \wedge \mathcal{T}$$

4. 对于拓扑绝缘体，$\mathcal{T} \wedge \mathcal{T}$ 项给出修正：
   $$\Delta C_1 = \frac{\tau^2}{4\pi^2} \int_{BZ} \text{Tr}[\mathcal{T} \wedge \mathcal{T}]$$ ∎

**数值估计**：

对于典型的拓扑绝缘体（如Bi₂Se₃），扭转强度与自旋-轨道耦合相关：

$$\tau \sim \frac{\lambda_{SO}}{E_g} \sim 0.1-0.3$$

因此扭转修正量级为：

$$\Delta C_1 \sim 10^{-3} - 10^{-2}$$

这在高精度测量中是可观测的。

### 3.3 拓扑不变量的分形推广

**定理3.3.1（分形Z₂不变量）** 在时间反演不变的系统中，分形拓扑不变量推广为：

## 4. 高温超导的分形机制

### 4.1 铜氧化物的分形结构

铜氧化物高温超导体（如YBCO、BSCCO）的晶体结构具有内在的层状分形特征：

**观察4.1.1（铜氧化物分形结构）** 铜氧化物中的CuO₂平面形成准二维的准晶结构，其缺陷分布表现出分形特征，分形维数 $d_f \approx 1.85$。

**分形结构来源**：
1. **氧空位有序**：氧原子在CuO链上的分布形成自相似结构
2. **条纹相**：电荷密度波的条纹图案具有分形特征
3. **孪晶边界**：正交相的孪晶结构形成分形网络

**定理4.1.1（分形电子态）** 铜氧化物中准粒子态在费米能级附近表现出分形特征，局域态密度满足：

$$N(r, E_F) \sim r^{d_f - d}$$

其中 $d = 2$ 为空间维数，$d_f$ 为分形维数。

### 4.2 配对对称性的扭转解释

高温超导的$d_{x^2-y^2}$波配对对称性可以从扭转几何的角度理解：

**定理4.2.1（配对对称性的几何起源）** 铜氧化物的$d_{x^2-y^2}$波配对源于CuO₂平面中氧位移导致的有效扭转场：

$$\Delta(\vec{k}) = \Delta_0 (\cos k_x - \cos k_y) \cdot f_{fractal}(\vec{k})$$

其中 $f_{fractal}(\vec{k})$ 为分形结构因子。

**推导**：

1. 在分形晶格上，BCS哈密顿量为：
   $$H = \sum_{k\sigma} \epsilon_k c_{k\sigma}^\dagger c_{k\sigma} + \sum_{k,k'} V_{k,k'} c_{k\uparrow}^\dagger c_{-k\downarrow}^\dagger c_{-k'\downarrow} c_{k'\uparrow}$$

2. 分形晶格上的配对相互作用获得几何因子：
   $$V_{k,k'} = V_0 \cdot \chi_{k-k'} \cdot \mu_{fractal}(k,k')$$

3. 在CuO₂平面中，最近邻和次近邻的相位积累导致：
   $$\chi_{k-k'} \propto (\cos k_x - \cos k_y)(\cos k'_x - \cos k'_y)$$

4. 这自然给出$d_{x^2-y^2}$波配对，分形因子调制配对强度 ∎

### 4.3 赝能隙与分形能谱

**定理4.3.1（赝能隙的分形解释）** 赝能隙态可以理解为分形能谱中的迁移率边现象，其特征能量与分形维数相关：

**物理图像**：

2. **多尺度涨落**：分形结构导致不同尺度的量子涨落，形成赝能隙
3. **相位涨落**：在$T_c$以上，库珀对仍存在（相位相干长度有限），导致赝能隙
